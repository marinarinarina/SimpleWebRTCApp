# WebRTC 1:1 P2P Communication

## Summary
1. Signaling Server with socket.io
2. Socket Event
3. MediaStream setup and RTCPeerConnection event
4. Send offer signal to other peer
5. Send answer signal to other peer

## How to Start (node)
```
# server
npm install
npm run start

# client
npm install
npm run dev
```
